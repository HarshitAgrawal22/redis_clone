| Your Constant        | RESP Prefix | Format                      | Example (Raw RESP)                   | Human Meaning           | Used For          |
| -------------------- | ----------- | --------------------------- | ------------------------------------ | ----------------------- | ----------------- |
| `error`              | `-`         | `-<message>\r\n`            | `-ERR unknown command\r\n`           | Error message           | Command failures  |
| `simple_string`      | `+`         | `+<string>\r\n`             | `+OK\r\n`                            | Simple success response | Acknowledgement   |
| `bulk_string`        | `$`         | `$<length>\r\n<string>\r\n` | `$5\r\nhello\r\n`                    | Binary-safe string      | Actual values     |
| `bulk_string (null)` | `$`         | `$-1\r\n`                   | `$-1\r\n`                            | Null value              | Missing key       |
| `integer`            | `:`         | `:<number>\r\n`             | `:100\r\n`                           | Integer number          | Counters, lengths |
| `array`              | `*`         | `*<count>\r\n<elements>`    | `*2\r\n$4\r\nHGET\r\n$4\r\nname\r\n` | List of elements        | Commands, lists   |
| `array (empty)`      | `*`         | `*0\r\n`                    | `*0\r\n`                             | Empty list              | Empty result      |
| `array (null)`       | `*`         | `*-1\r\n`                   | `*-1\r\n`                            | Null array              | Missing data      |
